# Docker#section06 : コンテナのライフサイクル

- docker run でコンテナ生成&コンテナ起動
- docker stop でコンテナ停止
- docker start でコンテナ起動
- docker attach で起動しているコンテナに接続
- docker rm でコンテナ削除
